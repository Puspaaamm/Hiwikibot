## ✨ Features

- 🌐 **Multilingual Support:** Search seamlessly using **Hinglish**, **Hindi**, or **English**.
- 🧹 **Clean Content Parsing:** Automatically removes unwanted Wikipedia references and external links, formatting headers neatly with custom symbols (`♠︎`).
- 🖼️ **Rich Media Delivery:** Fetches lead images, diagrams, and descriptive captions directly from Wikipedia.
- ⚡ **Chunked Messaging:** Automatically splits long articles into readable chunks to fit Telegram's message limits.
- 🕒 **24/7 Deployment Ready:** Built with a Flask background server designed to run continuously on free hosting platforms like Render.
